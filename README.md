# GeoPagos - Visa Solicitudes

Este proyecto es un gestor de aprobacion de solicitudes desde una landing page.

## Run
```
    npm install
    npm run start
```
Starts on `http://localhost:8080`

## Build
```
    npm run build
```

## Tecnologias de desarrollo
* React.js
* React Redux
* React Router 4
* React Hot Loader 3
* Webpack 3
* Babel
* Sass
* Bootstrap
* HTML5

# Test
```
    npm run test
```
